## 最基础

### 编译和运行

https://www.liaoxuefeng.com/wiki/1252599548343744/1255878730977024

1. 编写好源码，保存为Hello.java

2. javac Hello.java

   生成Hello.class

   注：若有中文注释或字符，则用javac -encoding UTF-8 Hello.java

3. java Hello

   执行Hello.class
